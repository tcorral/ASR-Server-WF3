# Open Text ASR Workflow Local Development environment
----------------------------------------------------------

## Dependencies:
1. NodeJS > 0.12.X
2. NPM > 3.X

## Install
### Clone this repository:
```
git clone https://github.com/tcorral/ASR-Server-WF3.git
```
### Navigate into the folder created by git
```
cd ASR-Server-WF3
```
### Install NPM dependencies
```
npm install
```
### Execute server
```
npm start
```
